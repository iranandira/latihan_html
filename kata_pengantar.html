Kata Pengantar

Buku dengan judul mudah dan cepat membuat program skripsi dengan VB 2012 merupakan buku teknik jitu membuat skripsi program dengan visual basic 2012. buku ini dapat anda pakai untuk membuat program atau tugas akhir untuk mahasiswa jurusan komputer atau informatika.Dengan materi yang saling berhubungan membuat anda dengan mudah belajar dari buku ini .

Buku ini membahas bagaimana membuat program skripsi untuk sistem pengajian dengan model prosedural, dan di buku berikutnya penulia akan menulis sistem pengajian dengan model objek. dalam buku ini penulis menggunakan database microsoft acces 2013, tetepi dengan format mdb.
